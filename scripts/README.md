# Scripts

This directory contains various utility scripts for the project.

## Database Emptying Scripts

- `empty-db.js`: A Node.js script to empty the database.
- `empty-db.sh`: A shell script (zsh compatible) to empty the database.

### How to Run

To run these scripts, navigate to the `scripts` directory and execute them directly:

```bash
cd scripts
./empty-db.js --help
./empty-db.sh --help
```

Ensure you have the necessary permissions to execute both scripts (`chmod +x empty-db.sh empty-db.js`).
